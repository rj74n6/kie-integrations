Privacy Policy

KIE Document Extractor — Custom GPT · Last updated: February 9, 2026

Summary: Documents you submit are processed in real time to extract the fields you request. We do not store your documents, extracted data, or conversation content after the request completes.

1. What this GPT does

KIE Document Extractor is a Custom GPT available through ChatGPT. When you upload a document (PDF or image) and describe what information you need, the GPT sends the document to the KIE extraction API at api.dillydally.dev and returns the extracted data to you in the ChatGPT conversation.

2. Data we collect

Data Collected? Stored? Purpose
Uploaded documents Transmitted No Processed in memory for extraction, then discarded
Extraction schema Transmitted No Defines which fields to extract
Extracted results Generated No Returned to you in the conversation
IP address / request metadata Yes Server logs (up to 30 days) Security monitoring, abuse prevention, debugging
Conversation content No No Handled entirely by OpenAI; we do not receive it

3. How documents are processed

  • Your document is base64-encoded by ChatGPT and sent to the KIE extraction API over HTTPS.
  • The API processes the document in memory only — no copy is written to disk or any database.
  • The extracted fields are returned in the API response and displayed in your ChatGPT conversation.
  • Once the HTTP response is sent, the document data is released from memory.

4. Data sharing

We do not sell, rent, or share your document contents or extracted data with any third party. Your data is transmitted between:

  • ChatGPT (OpenAI) — where the conversation takes place. OpenAI's privacy policy governs how they handle your conversation and uploaded files.
  • KIE extraction API — hosted at api.dillydally.dev, which processes your document and returns results.

5. Data retention

  • Documents and extracted data: Not retained. Discarded immediately after processing.
  • Server logs (IP addresses, timestamps, HTTP status codes): Retained for up to 30 days for security and debugging purposes, then automatically deleted.

6. Security

  • All data in transit is encrypted using TLS (HTTPS).
  • The API does not persist documents to disk or any external storage.
  • Access to server infrastructure is restricted to authorised personnel.

7. Your rights

Since we do not store your documents or extraction results, there is no personal data to access, correct, or delete on our side. For data held by OpenAI (your ChatGPT conversations), refer to OpenAI's privacy policy.

If you have questions about server logs or believe your data has been processed in error, contact us (see below).

8. Children's privacy

This service is not directed at children under 13. We do not knowingly process documents submitted by children.

9. Changes to this policy

We may update this policy from time to time. The "Last updated" date at the top of this page reflects the most recent revision. Continued use of the GPT after changes constitutes acceptance of the updated policy.
